# Bouge ta Fenêtre

Gestionnaire de fenêtres pour Windows, écrit en AutoHotkey v2.

![Version](https://img.shields.io/badge/version-1.5.0-blue)
![AutoHotkey](https://img.shields.io/badge/AutoHotkey-v2.0-green)
![Windows](https://img.shields.io/badge/Windows-10%2F11-blue)

## Fonctionnalités

- **Déplacer** les fenêtres au pixel près (1–500 px) avec Alt + Flèches
- **Redimensionner** avec Alt + Maj + Flèches
- **Centrer** sur l'écran actuel avec Alt + C
- **Changer d'écran** avec Alt + N
- **Toujours visible (TV)** : épingler une fenêtre au premier plan
- **Favoris** : sauver/restaurer la position d'une fenêtre (Alt + S / Alt + R)
- **Presets 1–4** : sauvegarder et replacer toutes les fenêtres d'un coup
- **Mini GUI** : interface compacte flottante, ancrable dans un coin
- **Volume** : ±1% / ±10% ou couper le son
- Liste des fenêtres ouvertes avec icône et suivi automatique

### Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| Alt + Flèches | Déplacer la fenêtre |
| Alt + Maj + Flèches | Redimensionner |
| Alt + C | Centrer sur l'écran |
| Alt + N | Écran suivant |
| Alt + S | Sauver position favorite |
| Alt + R | Restaurer position favorite |
| Alt + T | Basculer Always On Top |
